tsk

A simple CLI task tracker that stays out of your way.

No servers, no accounts, no configuration. Just a single binary and a JSON file.

go install github.com/zarldev/claude-mngr/cmd/tsk@latest
Get Started Commands Reference

Fast

Single binary, no startup overhead. Add a task in milliseconds.

Simple

Four commands: add, list, done, rm. Nothing else to learn.

Portable

Tasks stored in ~/.tasks.json. Easy to sync, back up, or inspect.

Zero Dependencies

Pure Go, no external services. Works offline, works everywhere Go runs.

Quick Start

$ tsk add Buy groceries
added task 1: Buy groceries

$ tsk add Write tests for auth module
added task 2: Write tests for auth module

$ tsk list
  1  [ ]  Buy groceries                            just now
  2  [ ]  Write tests for auth module               just now

$ tsk done 1
task 1 marked done

$ tsk list
  1  [x]  Buy groceries                            2m ago
  2  [ ]  Write tests for auth module               2m ago
